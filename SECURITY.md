# Security Policy

## Supported Versions

Only the latest release version of Cisco Switch Docu-Crawler is actively supported with security updates. We encourage all users to run the latest version from the main branch.

| Version | Supported          |
| ------- | ------------------ |
| >= 1.0  | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of our tool seriously. If you find a security vulnerability, please do **not** open a public GitHub issue. Instead, report it privately to the maintainer:

* **Email**: Send detailed vulnerability details to **leifdavisson@gmail.com**

Please include:
* A description of the vulnerability and potential impact.
* Steps to reproduce the issue (proof-of-concept script/exploit if applicable).
* Details of the test environment (OS version, Python version, target device model).

We will acknowledge receipt of your report within 48 hours and work on a prompt patch. Once resolved, we will publish the fix and credit you for the disclosure if desired.
